# Backend Service
   
   ## Tech Stack
   - Python 3.10+
   - FastAPI
   - Neo4j
   - Redis
   - Kafka
   
   ## Setup
```bash
   cd backend
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
```
   
   ## Run
```bash
   uvicorn src.main:app --reload
```
